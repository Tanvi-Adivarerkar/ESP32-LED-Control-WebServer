# ESP32 LED Control Web Server

An IoT-based project that uses ESP32 to control multiple LEDs through a web interface over a Wi-Fi network.

## Features

* Remote LED control using Wi-Fi
* Web-based control panel
* Control multiple LEDs independently
* Real-time ON/OFF switching
* ESP32 hosted web server

## Technologies Used

* ESP32
* Arduino IDE
* WiFi Library
* HTML
* Embedded Systems

## Project Structure

* Ledcontrol_webserver.ino → Main Arduino code
* LEDcontrol_webserver_documentation.docx → Project documentation
* screenshots/ → Output screenshots

## How It Works

1. ESP32 connects to a Wi-Fi network.
2. A web server is hosted on the ESP32.
3. User opens the ESP32 IP address in a browser.
4. A web page with LED control buttons is displayed.
5. LEDs can be turned ON or OFF remotely.
